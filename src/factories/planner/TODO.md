# Floor Planner - Future Features

## High Priority

### 1. Conveyor Belt Connections

- [ ] Draw conveyor belt connections between building inputs/outputs
- [ ] Show conveyor belt lengths and paths
- [ ] Auto-routing between buildings
- [ ] Support for conveyor lifts at different heights (track height in node data)

### 2. Building Input/Output Configuration

- [ ] Allow user to configure inputs/outputs based on planned production recipes
- [ ] Display recipe information on buildings
- [ ] Show throughput rates on connections
- [ ] Validate that input/output types match when connecting

### 3. Save/Load Floor Plans

- [ ] Save floor plans with factory data
- [ ] Load floor plans when opening a factory
- [ ] Export floor plans as images or JSON
- [ ] Import floor plans from saved files

## Medium Priority

### 4. Enhanced Building Details

- [ ] Show power consumption
- [ ] Display clock speed settings
- [ ] Show resource production rates
- [ ] Building status indicators (powered, producing, idle)

### 5. Multi-Level Support

- [ ] Add floor/level selector
- [ ] View multiple floors stacked
- [ ] Conveyor lifts connecting between floors
- [ ] Foundation height tracking

### 6. Advanced Layout Tools

- [ ] Alignment guides and snapping
- [ ] Multi-select and group operations
- [ ] Distribute evenly tool
- [ ] Mirror/flip selections
- [ ] Ruler and measurement tools

## Low Priority

- [ ] Building images/sprites instead of colored rectangles
- [ ] Animated conveyor belts showing item flow
- [ ] Glow effects for powered buildings
- [ ] Dark mode support

### 8. Collaboration Features

- [ ] Share floor plans with other users
- [ ] Collaborative editing
- [ ] Comments and annotations
- [ ] Version history

### 9. Performance Optimization

- [ ] Virtual rendering for large factories
- [ ] Layer-based rendering
- [ ] Optimize node rendering performance

### 10. Integration Features

- [ ] Auto-generate floor plans from calculator results
- [ ] Export to game save files (if possible)
